# 🚢 Python Battleship Game

A console-based implementation of the classic **Battleship** game developed in **Python** as part of a collaborative software development challenge.

This project focuses on game logic, modular programming, teamwork, and problem-solving through the recreation of one of the most popular strategy games.

---

## 🎯 Project Overview

Battleship is a turn-based strategy game where players attempt to locate and sink their opponent's fleet by selecting coordinates on a grid.

This project was developed collaboratively, with team members contributing to different components of the game architecture, including board generation, game mechanics, and supporting functions.

---

## 📂 Project Structure

```text
python-battleship-game/
│
├── main.py
├── tablero.py
├── funciones.py
├── variables.py
└── .gitignore
```

* **main.py** → Game execution and flow.
* **tablero.py** → Board creation and management.
* **funciones.py** → Helper functions and game mechanics.
* **variables.py** → Configuration variables and constants.

---

## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/Juditga97/python-battleship-game.git
```

Navigate to the project folder:

```bash
cd python-battleship-game
```

Run the game:

```bash
python main.py
```
